```

2. IP Catalog → search "PIO" → double-click
3. Name it: fsm_status_pio
4. Settings: 8-bit Input, Address 0x6000
5. Finish, then repeat for timer_status_pio @ 0x7000
6. Generate HDL → Generate → Close
```

## 📞 TROUBLESHOOTING

A: Did you add PIOs to soc_system.qsys? Run `QUICK_VERIFY.ps1`

**Q: Buttons don't work?**  
A: Check HPS can read 0x6000 register with `devmem2 0xFF206000`

**Q: Timer doesn't count down?**  
A: Check idle_timer.v line 57 has the fix

**Q: LCD shows garbage?**  
A: LCD driver issue, not related to ULTRA PLAN fixes
