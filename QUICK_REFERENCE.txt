## WHAT EACH COMPONENT DOES

```
KEY[0-3] → Debouncer(50ms) → EdgeDetect → Idle Timer + Verilog FSM → HEX Display
                                                   ↓
                                              HPS Software
                                    Reads: 0x6000, 0x7000
                                    Renders: LCD only (FSM stays in FPGA)
```

## TROUBLESHOOTING

Q: Compilation fails?
A: Run QUICK_VERIFY.ps1 and inspect Quartus messages first.

Q: Buttons do not affect LCD?
A: Check HPS can read 0x6000 register and verify FSM state changes.

Q: Timer does not count down?
A: Verify timer register at 0x7000 and confirm reset path on button pulses.

Q: LCD shows garbage?
A: Confirm LCD initialization and SPI wiring/software driver path.
